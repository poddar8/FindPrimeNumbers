# FindPrimeNumbers
Simple java program to find prime numbers
</BR></BR>Usage
</BR>Command to execute in terminal
    </BR><P>java -jar FindPrimeNumber.jar <any integer>
</BR></BR>Example
    <P>java -jar FindPrimeNumber.jar 171 </P>
